# 骰评 · Economic Atlas

> Every Reality Holds More Than One Truth.

🎲 **骰评·经济版**是一台经济解释采样引擎。输入一个经济问题或现象，系统首先生成**冲突轴**（结构张力识别），然后在 10 个理论透镜的视角下，通过两轮 AI 调用输出一篇多视角结构化的经济评论。

它不是经济新闻摘要。它是**多理论解释空间中的因果路径采样器**——同一个话题，重新掷骰可能选中地缘经济学、凯恩斯主义或世界体系理论，给出完全不同的因果链。

## V2 四层架构

```
经济现象
  ↓
L1: 冲突轴识别                              ← 成本转移 vs 政策吸收
      2-3 条张力对 + 每轴 2-3 个具体机制
  ↓
L2: 理论透镜（10 个）                       ← 制度经济学/地缘经济/世界体系/马克思/凯恩斯…
      翻译规则强制锚定：每个理论必须从具体机制出发
  ↓
L3: 两轮 AI 采样                            ← 第一轮选理论+因果链，第二轮写正文
      禁止概念填空（"金融作用于周期"）
  ↓
L4: 输出                                    ← 冲突轴 + 骰子路径 + 正文 + 竞争性观点 + 金句收束
```

## 核心特性

### 10 个理论透镜（V2 扩展）

制度经济学 | 地缘经济学 | 世界体系理论 | 马克思主义政治经济学 | 凯恩斯主义 | 依附理论 | 新自由主义批判 | 调节学派 | 现实主义政治经济学 | 行为/叙事经济学

每个理论附带翻译规则——当用作分析时，必须锚定在具体机制上，不能停留在抽象概念。

### 冲突轴（取代 9 格分类）

识别核心结构张力对，如：
- 规则主导权 vs 发展空间
- 成本转移 vs 政策吸收
- 产业升级 vs 外部约束

### 22 条时空经济体数据库

覆盖 1929-2025 年 7 大经济体，按国家战略、金融危机、货币体系、政策实验等分类。匹配时自动调整结构场权重和理论适配度。

### 三层匹配逻辑

数据库精确匹配 → 关键词启发式 fallback → 纯随机兜底

### 评论文本与分享卡片分离

正文完整展示在页面上，分享卡片精简为冲突轴 + 理论透镜 + 骰子路径 + 洞察 + 摘要，适合社交媒体传播。

### 写作规则（8 条）

1. 机制锚定 — 第一句不可替换
2. 翻译强制 — 禁止概念填空，每步可验证
3. 理论后置 — 先描述现象再用理论
4. 反观点 — 认真对待反面论证
5. 多玩家意识 — 不把经济体当单一体
6. 收束金句 — 短促有力，可被独立引用
7. 因果链一致性 — 正文与链一一对应
8. 竞争性视角 — 2 个其他理论的对抗解释

## 快速开始

```
打开 → 输入经济问题 → 选择 AI 引擎 → 填入 API Key → 生成 → 再掷
```

模板引擎无需配置可直接体验（质量较低）。AI 引擎需要 DeepSeek / OpenAI / Ollama API Key。

## 部署

纯静态，两个文件：

```
index.html        # 主应用
economy_db.js     # 22 条时空经济体数据库
```
