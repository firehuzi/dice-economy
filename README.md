# 骰评 · Economic Atlas

> Every Reality Holds More Than One Truth.

🎲 **骰评·经济版**是一台经济解释引擎。输入一个经济问题或现象，系统首先生成 9 大结构场分解，然后在 5 个理论框架的视角下，通过数据库加权的骰子采样，输出一篇多视角结构化的经济评论。

它不是经济新闻摘要。它是**多理论解释空间中的因果路径采样器**——同一个经济现象，马克思主义、凯恩斯主义、制度经济学和行为经济学各自给出完全不同的因果链。每掷一次，就是换一个角度重新解释现实。

## 四层架构

```
经济现象
  ↓
L1: 结构场（9 格经济分解）         ← 生产/分配/需求/价格/金融/制度/叙事/周期/外部
  ↓
L2: 解释层（5 个理论框架）         ← 马克思/新古典/凯恩斯/制度经济学/行为经济学
  ↓
L3: 骰子引擎（数据库加权采样）      ← 21 条时空经济体档案加权影响骰子走向
  ↓
L4: 输出（结构化评论）              ← 开场→主干→对照→冲突→收束
```

## 核心特性

### 21 条时空经济体数据库

| 类型 | 条目 |
|---|---|
| 国家战略/地缘经济 | 马歇尔计划、一带一路、日本/韩国/中国奇迹 |
| 金融危机/泡沫 | 大萧条、次贷、日本泡沫、欧债、亚洲风暴 |
| 货币体系 | 布雷顿森林、广场协议、欧元 |
| 结构转型 | 四小龙、苏东剧变 |
| 供给冲击 | 石油危机、新冠 |
| 政策实验 | 特朗普1.0/2.0、英国脱欧 |
| 房地产 | 中国房价 |

### 数据库加权

输入匹配到数据库条目后，结构场权重和理论适配度自动调整。同一问题在不同时空背景下生成完全不同的分析路径。

### 双引擎模式

- **模板引擎**：零配置，即时出结果
- **AI 引擎**：DeepSeek / OpenAI / Ollama

## 快速开始

```
打开 → 输入经济问题 → 生成 → 再掷
```

模板引擎无需配置。AI 引擎需要 API Key。

## 部署

纯静态 HTML，部署到任何静态托管即可。两个文件：

```
index.html        # 主应用
economy_db.js     # 21 条时空经济体数据库
```
